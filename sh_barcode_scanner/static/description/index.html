Sale Barcode Scanner

Products With EAN-13 Barcode.

Product scan option in sale settings whether you want to scan product by Barcode,QRCode, internal reference or all. Note: Move to top feature is workable in edit mode, so first you need to save record.

Scan barcode then product automatic added in order line if product already existed in order line then ordered qty increased by 1.

Error message given if no any matching product found.

Error message given if sale order in cancelled or locked state.

Products With QRCode.(Note: QRCode workflow remains the same everywhere.)

Product with QRCode in form view.

Select QRCode if you want to scan using only QRCode.

Scan QRCode then product automatic added and if the product already existed then ordered qty increased by 1.

Inventory Barcode Scanner

Products With EAN-13 Barcode.

Product scan option in inventory settings whether you want to scan product by Barcode,QRCode, internal reference or all. Note: Move to top feature is workable in edit mode, so first you need to save record.

Error message given if you enable lots and serial number.

when you Scan barcode in detailed operation wizard if no line added then error message will show.

if you scan barcode then quantity done increased by 1.

Error message given if picking not in proper state.

warning message given if done quantity increased than initial demand.

error message given if scanned product not found in line.

Scrap Barcode Scanner

Products With EAN-13 Barcode.

Product scan option in inventory settings whether you want to scan product by Barcode,QRCode, internal reference or all.

Scanned product added in product field and quantity set to 1 if product already in product field then quantity increased by 1.

Scan barcode then product automatic added in product field and quantity set to 1 if product already in product field then quantity increased by 1

Error message given if you try to change product in scrap order after scanning started.

Error message given if scrap order in Done state.

Error message given if no any matching product found.

Inventory Adjustment Barcode Scanner

Products With EAN-13 Barcode.

Product scan option in Inventory settings whether you want to scan product by Barcode,QRCode, internal reference or all.

Press "Start Inventory Scanning" to scan with a barcode scanner.

We have added tree view in the form view. You can scan products and that counted qty increase by 1(One).

The error message is given if inventory adjustment is a draft or validated state.

The error message is given if no matching product found.
